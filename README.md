La página web desarrollada corresponde a una landing page sobre la modalidad de Programación y Diseño Web. Durante el avance del proyecto se implementó la estructura principal utilizando HTML y se aplicaron estilos visuales mediante CSS para mejorar la presentación y organización del contenido.

En la parte superior de la página se creó un banner principal con una imagen de fondo que ocupa todo el ancho de la pantalla, permitiendo una presentación visual más atractiva. Después del banner se añadieron un título principal y un subtítulo centrados para identificar claramente el tema de la página y la sección correspondiente.

Posteriormente, se incorporó un párrafo descriptivo donde se explica en qué consiste la modalidad de Programación y Diseño Web, mencionando herramientas como HTML, CSS, JavaScript y conceptos relacionados con el diseño y desarrollo de páginas web responsivas.

En el cuerpo principal de la página se diseñó una sección con tres tarjetas visuales utilizando contenedores <div>. Cada tarjeta contiene una imagen de fondo relacionada con la temática del proyecto. Para organizar estas tarjetas se utilizó la propiedad display: flex en CSS, permitiendo una distribución uniforme y ordenada.

Finalmente, se desarrolló un pie de página (footer) dividido en tres secciones: contacto, servicios y tecnologías. Allí se incluyó información básica como correo electrónico, número telefónico, servicios ofrecidos y tecnologías utilizadas en el desarrollo web.

En cuanto al diseño visual, se aplicaron estilos CSS para personalizar colores, tamaños, tipografías, alineación de texto e imágenes de fondo. También se trabajó en la organización de espacios y márgenes para lograr una interfaz más limpia y agradable visualmente.
